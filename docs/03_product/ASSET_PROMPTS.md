# Asset Prompts

These prompts are prepared for later production asset generation. They are not
implementation requirements for the current surface-navigation migration.

## Building Surface Frame

Create an integrated Roman comic-book game UI frame for a building management
surface. The frame sits inside an existing game shell, not as a centered modal.
Use dark bronze, parchment, gold accents, ink-like borders, a first-level list
state and a second-level selected building detail state with a hero area and a
wide content workspace. Keep the
surrounding ludus courtyard visible. No blur overlay, no SaaS dashboard styling,
no watermark.

## Building Skill Tree

Create a Roman comic-book skill tree UI for building upgrades. Include four
tier groups, connected bronze lines, purchased nodes with green check marks,
available nodes with gold highlights and locked nodes with dark chain styling.
Use parchment cards, readable icons, compact effect previews and a right-side
purchase inspector with cost, requirements, effects and a green CTA. No neon,
no sci-fi, no table layout.

## Contextual Gladiator Sheet

Create a compact contextual gladiator sheet attached to a building surface.
Include a memorable portrait, name, level badge, condition/status badges,
compact attributes, XP progress, skill bars and one primary CTA to open the full
gladiator profile. The sheet should feel pinned to the current surface, not like
a modal. Use parchment, bronze trim, red alert accents and green CTA treatment.

## Roster Register

Create a master-detail gladiator roster register for a Roman management game.
Left side: scannable rows with portraits, status, level and compact stats.
Right side: selected gladiator profile with portrait, progression, planning and
combat record. Keep the style warm, comic-book, parchment and bronze, with
strong silhouettes and no modern dashboard tables.

## Lanista Planning Table

Create a full-screen weekly planning surface styled as a lanista's table with
parchment, bronze tools and readable day columns. Include week projections,
activity cards, risk indicators, point allocation controls and a green primary
CTA. The surface remains inside the game shell and should preserve Roman comic
management-game readability.

## Reusable UI Texture Set

Create reusable material references for dark bronze HUD panels, parchment
surfaces, comic ink borders, alert badges, resource chips and green CTA buttons.
Keep colors warm and readable: bronze, charcoal, parchment cream, gold, red
banners, green action states and small sky/olive accents.
